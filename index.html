TLCSharp

Unofficial Telegram (http://telegram.org) client library implemented in C# on .NET Core.

This project is based on TLSharp Project by sochix

What's new

  • .NET Core 2.1
  • Multiuser
  • Custom Session Path
  • SendBotButton (TLRequestGetBotCallbackAnswer)
